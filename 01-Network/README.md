# Network Infrastructure

This section documents the network architecture used in the Enterprise IT Infrastructure Deployment project.

## Components

- TP-Link ER605 Router
- TP-Link SG2218P Managed Switch
- 3 × TP-Link EAP620 HD Access Points

## VLAN Configuration

### VLAN 10 - Corporate

Used for:

- Domain Workstations
- Windows Server
- Printer
- Attendance System
- Network Infrastructure

### VLAN 20 - Guest

Used for:

- Mobile Devices
- Visitors
- Guest Wi-Fi

## Security

- Inter-VLAN Isolation
- Centralized Management via Omada Controller
